# Quantum Tunneling Simulation Codes

This folder contains the computational codes developed for the paper:

Montiel-Purdon, F. Y., Villavicencio, J., & Hernández, A. (2025).
"Larmor and Delay Times in Transient Quantum Tunneling."
Physica Scripta, 100(6). URL: https://iopscience.iop.org/article/10.1088/1402-4896/add220/meta

## Description

The codes implement numerical simulations related to transient quantum tunneling in semiconductor systems under magnetic field effects.

The repository includes:
- Numerical implementation of Moshinsky functions
- Probability density calculations
- Spin expectation value simulations
- Larmor time analysis
- Dynamical characteristic time calculations
- Data visualization and figure generation

## Tools and Libraries

- Python
- NumPy
- SciPy
- Matplotlib
- Pandas
- SymPy

## Purpose

These codes were developed to:
- Analyze transient quantum tunneling dynamics
- Study magnetic field effects on tunneling processes
- Compute characteristic tunneling times
- Generate numerical results and publication figures

## Notes

The codes are research-oriented and focused on scientific computing, numerical analysis, and computational modeling in quantum systems.
