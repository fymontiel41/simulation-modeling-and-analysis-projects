# Estabilidad de Vacios de Sitter 

Este repositorio contiene el código desarrollado para el cálculo numérico del vector de períodos en geometrías Calabi–Yau de intersección completa y el análisis de estabilidad de vacíos de Sitter provenientes de potenciales efectivos derivados de supergravedad tipo IIB. En particular, la (4,1)-variedad del artículo: https://arxiv.org/pdf/1512.08367. El código implementa algoritmos de cómputo simbólico y análisis numérico mediante Wolfram Mathematica para evaluar potenciales efectivos, estudiar la estabilización de módulos y explorar distintas configuraciones geométricas dentro del espacio de parámetros considerado. Asimismo, se automatiza la organización y análisis de resultados numéricos obtenidos a partir de las simulaciones y evaluaciones realizadas.
